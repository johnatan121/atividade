# calculo imc. 
## objetivo.
realizar o calculo de imc.
## funcionalidade.
atraves dos dados infromado pelo usuario como altura, peso e idade o programa ira realiza o calculo e retrona com a infromação de Índice de Massa Corporal  
### Baixo Peso	< 18,5  
### Peso Normal	18,5 a 24,9      
### Excesso de Peso	25 a 29,9  
### Obesidade Grau I	30 a 34,9  
### Obesidade Grau II	35 a 39,9  
### Obesidade Mórbida	≥ 40
